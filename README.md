# Gaming Coin Hub - Preview

## Dynamic Content

Edit **`js/site-data.js`** to update content across the website:

- **nav** – Navigation links
- **footerLinks** – Footer links
- **services** – Core Infrastructure cards (index)
- **packages** – Sweepstake platform tiers (Bronze, Gold, Diamond)
- **marketplace** – Add-on services
- **features** – Why Us section
- **marqueeGames** – Games shown in the hero marquee
- **gameNames** – Full list of supported games (28)

## Structure

- **index.html** – Landing (Services, features, CTA). Packages & Marketplace moved to sweepstake.
- **sweepstake-websites.html** – Packages, Marketplace, games list
- **gaming-coins.html** – Gaming coins + games grid
- **virtual-numbers.html** – Virtual numbers
- **ip-proxies.html** – IP proxies
- **game-rates.html** – Game rates table (Player/Admin links)
- **admin.html** – Admin login, add games
